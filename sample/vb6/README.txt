This sample project provides a quick demo of how to use various functions

It will tell you what the remote node likes to be called (its Node Identifier)

Running this assumes that you have first compiled libxbee.dll successfully
You will also need to either copy libxbee.dll into this folder

If you want to use libxbee in your own projects, you must include libxbee.bas
  which will provide you with access to the functions and type declarations
